위 사진의 플러그인을 설치 후 작업진행하면 됩니다.

scss의 기본적인 사용법은 

예) body .section .continer .title{ 
  font-size: 36px;
}
위 처럼 적었던 css를

body{
  .section{
    .container{
      .title{
        font-size: 36px;
      }
    }
  }
}
이렇게 적으면 됩니다.
